VidMix Web

VidMix — fast browser video/audio mixing with optional server fallback.

Mix video and audio locally in the browser using WebAssembly.

Server mode requires `server.py`
Browser mode requires WebAssembly and file APIs. Best experience is in Chrome, Firefox, or Edge. Safari support is improving but may be slower and memory-limited.
